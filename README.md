# Tenzor Tests

Тестовое задание на позицию разработчика в тестировании (Программист-тестировщик).

## Установка

git clone https://github.com/Iskandar54/Tenzor_tests.git

python -m venv .venv

.\.venv\Scripts\activate

pip install -r requirements.txt

## Запуск тестов

pytest
